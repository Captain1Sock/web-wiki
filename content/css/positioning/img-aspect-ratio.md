## Maintain aspect ratio of a div
```
aspect ratio  | padding-top value
--------------|----------------------
    1:1       |       100%
    16:9      |       56.25%
    4:3       |       75%
    3:2       |       66.66%
    8:5       |       62.5%
```

<br />

```css
elemenet {
    padding-top: %value;
}
```
